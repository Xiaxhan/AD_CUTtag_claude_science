# AD CUT&Tag Analysis with Claude Science

This repository contains the analysis workflow, code, selected figures, and summarized results generated for the **Built with Claude: Life Sciences Hackathon**.

Please review the summary in `output/REPORT.md`

## Project overview

This project uses single-nucleus CUT&Tag data from Alzheimer’s disease (AD) brain samples to characterize cell-type-specific epigenomic remodeling.

The analysis demonstrates how Claude Science can support biological interpretation, experimental target prioritization, integration of chromatin and gene-expression evidence, and development of a mechanistically focused research proposal.

## Repository structure

* `scripts/`: analysis and visualization code
* `output/`: selected reports, figures, and summarized results
* `Project.md`: project instructions, analysis plan, and Claude Science workflow

## Key outputs

1. **Complete analysis summary:** [`output/REPORT.md`](output/REPORT.md)  
   Documents the full analysis workflow, results, and initial biological interpretation.

2. **Critical review and target prioritization:** [`output/AD_CRE_story_summary.md`](output/AD_CRE_story_summary.md)  
   Refines the biological interpretation and prioritizes candidate genes and regulatory elements for experimental validation.

3. **Grant proposal strategy:** [`output/grant_strategy_memo.md`](output/grant_strategy_memo.md)  
   Translates the findings into a mechanistically focused research question, specific aims, experimental strategy, and testable predictions.
   
## Data availability

The publicly available single-nucleus CUT&Tag data used in this project were downloaded from Synapse under accession **syn53191971**:

https://www.synapse.org/Synapse:syn53191971/wiki/632636

Large input datasets, intermediate files, cache files, and RDS objects are not included in this repository.
